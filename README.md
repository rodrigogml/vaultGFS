# vaultGFS

Generic, configurable backup utility with GFS-style filesystem backups, MySQL logical dumps, manifests, SQLite catalog and cron-line scheduling.

Executables:

- `vaultgfs-backup`: runs backup jobs.
- `vaultgfs-reload`: validates config and syncs cron-line schedules.
- `vaultgfs-restore`: future restore helper.

See `docs/requirements.md`.
